Mi Guía Práctica de HTML

Aprender las etiquetas básicas experimentando con ellas.

Este es un título (H2)

Este es un subtítulo (H3)

<h1> a <h6>

Sirven para definir la jerarquía de los títulos de la página. El H1 es el título principal, mientras que del H2 al H6 se usan para secciones menores.

Este es un texto de párrafo común y corriente en una página web.

<p>

Significa "párrafo". Se utiliza para estructurar bloques de texto normales. Añade automáticamente un espacio en blanco vertical para separar los bloques.

Ir a GitHub
<a>

Crea hipervínculos. Usa el atributo href para la dirección de destino. El atributo target="_blank" fuerza al navegador a abrirlo en otra pestaña.

Bloque Div
<div>

Es una caja vacía invisible. Sirve como contenedor genérico para agrupar elementos y estructurar el diseño o aplicarles estilos visuales en bloque.
